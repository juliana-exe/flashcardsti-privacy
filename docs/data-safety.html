Detalhes sobre Dados Armazenados Localmente

Progresso de estudo

O aplicativo armazena localmente no seu dispositivo quais flashcards você marcou como "Sei" ou "Revisar". Esse dado:

  • Nunca sai do dispositivo — não é transmitido pela Internet.
  • Não é acessível por nós — não temos acesso aos seus dados.
  • É apagado ao desinstalar o aplicativo.
  • Pode ser deletado manualmente via botão de reset dentro do app.

Práticas de Segurança

Criptografia dos dados em trânsito

Não aplicável — o app não transmite nenhum dado pela rede.

Exclusão de dados

Você pode deletar todo o seu progresso usando o botão "Resetar" dentro do aplicativo, ou simplesmente desinstalando o app. Não há dados armazenados em servidores externos para excluir.

Permissões do aplicativo

O FlashCards TI não solicita nenhuma permissão especial do dispositivo. Não requer acesso a câmera, microfone, contatos, localização, armazenamento externo ou qualquer outra permissão sensível.
